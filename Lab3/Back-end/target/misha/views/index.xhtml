Web Lab 3

Web programming lab 3

Develop an application based on JavaServer Faces Framework that checks if the point hits the specified area on the coordinate plane.

The application must include 2 facelet templates - the start page and the main page of the application, and also a set of managed components that implement server-side logic.

The start page should contain the following elements:

  • "Header" containing the student's full name, group number and option number.
  • An interactive clock showing the current date and time, updated every 1 seconds.
  • The link that will take you to the main page of the application.

The main page of the application should have the following elements:

  • A set of components for defining point coordinates and area radius in accordance with the specification.
  • Dynamically updated image representing the area on the coordinate plane according to the number options and points whose coordinates were specified by the user.
  • A table with the results of previous checks.
  • Link to return to the start page.
